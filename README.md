# GW2 Wiki Mirror (镜像站)

这是一个为《激战2》(Guild Wars 2) 官方 Wiki 搭建的镜像代理服务，旨在解决部分地区访问官方 Wiki 困难或速度较慢的问题。

## 🚀 项目架构
本项目基于 **Vercel Edge Functions** 构建，通过边缘网络代理流量，绕过了部分云服务商的 IP 访问限制。

* **源站**: [wiki.guildwars2.com](https://wiki.guildwars2.com)
* **代理节点**: Vercel Global Edge Network
* **DNS 管理**: Cloudflare



## 🛠️ 技术特点
* **高可用性**: 利用全球边缘节点，动态请求官方 Wiki 数据。
* **重定向修复**: 自动拦截并修正官方 Wiki 的强制跳转，确保页面始终在镜像域名内浏览。
* **内容替换**: 自动识别并替换页面中的官方链接，防止用户无意间跳出镜像站。
* **智能缓存**: 配合 Cloudflare 缓存策略，在提升访问速度的同时大幅节省流量额度。
